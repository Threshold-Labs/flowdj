# FlowDJ — Features

See **[PRD.md](./PRD.md)** for the full product requirements document including shipped features, upcoming phases, architecture, and open questions.

## Quick Summary

**Shipped (v0.2):**
- Generative audio engine with 6 distinct sound palettes (FM synthesis, delay, LFO, swing, extended harmony)
- Flow tracking from typing speed + voice/mic energy
- Project focus system with musical drift response (11 real project profiles)
- Voice commands (Chrome/Edge), responsive UI, single-tab enforcement

**Next up:**
- Phase 1: Mobile & on-the-go (motion input, PWA, voice-only mode)
- Phase 2: Attention telemetry & dashboard
- Phase 3: Spotify integration (taste profiles, real track mixing)
- Phase 4: External attention sources (git, calendar, active window)
- Phase 5: Social / multiplayer
- Phase 6: Advanced audio (on-device AI models, spatial audio, circadian rhythm)
